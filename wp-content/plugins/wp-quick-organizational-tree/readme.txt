Add a site map on any of your page using the simple shortcode. Auto generator of a customizable and designed sitemap page to improve the navigation of your website

== Description ==
Tree Website Map is a plugin that automatically generates a site map page from your WordPress site. Easy way to add a website map on one of your pages becomes reality thanks to this WordPress plugin. Just use the shortcode on any of your pages. This will automatically generate a sitemap of all your pages and posts,
<br>
<strong>Tree Website Map includes: </strong>
<ul>

<li>Dynamically create your website tree for all posts &amp; categories</li>
<li>Ability to exclude any post or category</li>
<li>Ability to hide/show post & category thumbnail</li>
<li>Support shortcode</li>
<li>Display all pages, posts and categories</li>
<li>Display the Custom Post Type</li>
<li>Posts and categories displayed hierarchically</li>
<li>Display the Custom taxonomies</li>
<li>Fully customized and easy to understand</li>
<li>Possibility to exclude some pages or some Custom Post Type</li>
<li>Ajax loading</li>
<li>Fully customized and easy to use</li>
</ul>


== Installation ==
1. Download the package.
2. Extract the contents of .zip folder to wp-content/plugins/ folder or upload it using your pligins page > add new > upload
3. Activate the Plugin.
4. Go to your main menu > Website map tree.

Thanks!
