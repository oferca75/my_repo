== Installation ==

1. Upload st-categories-wp folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to widgets page
4. Drag and drop the widget where ever you need
5. Choose the parent category to display the child categories in the front end

 - Please Find "ST Categories WP" in settings menu of your WordPress admin panel. Select "Document" in "Abouts" Table to  views Plug-in user Guides
